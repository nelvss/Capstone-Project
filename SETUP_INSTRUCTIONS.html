1 Configure Email Credentials

‚ö†Ô∏è For Gmail Users: You must create an App Password:
  1. Go to Google Account Security
  2. Enable 2-Step Verification
  3. Click App passwords at the bottom
  4. Select app: Mail, Device: Other
  5. Copy the 16-character password (no spaces)
  6. Paste it as EMAIL_PASSWORD in .env file
2 Test Email Configuration

Run this command to verify your email settings work correctly:

cd my-express-web
npm run test:email

This will send a test email to yourself. Check your inbox to confirm it worked!

3 Start the Email Server

Start the Node.js server that handles email sending:

cd my-express-web
npm start

Or use the PowerShell script:

.\start-server.ps1
